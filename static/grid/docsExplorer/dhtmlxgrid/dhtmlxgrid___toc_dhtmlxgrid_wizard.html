dhtmlxGrid Wizard

dhtmlxGrid Wizard is a utility that provides an easy way to create grids, set and view their layouts and change them in accordance with user's specific needs with practically no coding from the user's side. This utility has a user-friendly interface and it is really powerful and easy to use. It will be helpful not only for beginner developers but also for professionals who want to save their time on configuring.

http://dhtmlx.com/docs/products/dhtmlxGrid/doc/wizard/index.html

This utility provides means for users to:

  • Create custom grids with the unlimited number of columns;
  • Manipulate with grid settings, columns settings and grid style;
  • Set the type of grid configuration: via JavaScript or via XML;
  • View all the changes applied to the grid right away in the grid and in the code;
  • Easy implement the newly-created grid in a user's project - just copy the grid coding;
  • Enable/disable modes for grid, e.g.smart rendering mode.
