dhtmlxgrid

Excells with Special Purpose

sub_row

It shows the cell as an expandable sub row of the grid row (allows to implement “expandable details” pattern).

Variation:

  • sub_row_ajax - data in the cell is considered to be an URL for the ajax request.
  • sub_row_grid - allows to create a nested sub grid as content of the sub_row.
