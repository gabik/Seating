CodeThatGrid Objective

CodeThatGrid is an advanced JavaScript grid looking like Microsoft Excel grid thatthat enables user to display in efficient and fancy way the massifes of data from variety of sources including the csv and database. Fully customizable through the CSS the grid is the definite answer to the numerous cases when structured data has to be displayed.

standard version is FREE for educational or non-commercial use

is an advanced cross-browser state-of-art JavaScript table

can be easily integrated in any web page

requires no JavaScript knowledge from the prospective user

provides with the wide range of the features some of which are unique and not to be found elsewhere

comes in two editions STANDARD and PRO.  Read Standard vs PRO to know the differences between CodeThatGrid Standard and PRO.

CodeThatGrid Features Set

The following positions and features are supported by the grid script:

Features specific for CodeThatGrid only (missed at CodeThatTable):

Heading buttons (active and passive)

Multiple row and column selection, with mouse dragging support

Cell range selection

Support of the XML and CSV files reading

Paste content directly from the clipboard. For example copy a region in excel, and paste it here using the button or ctrl-v.

Inserting columns and rows

Deleting columns and rows.

Resizing columns and rows

Vertical and horizontal scrolling capabilities

Text

Email, URL

Image

Row, column and cell colors

Row, column and cell alignment

Alternating rows colors, on-mouse-over row effect and easy row (un)marking with mouse click

Each column has its own settings for data type, caption, etc.

Row, column and cell word wrapping (PRO version only)

Multi- row and column selection, with mouse dragging support (PRO version only)

Sorting and multi-sorting features (ascending and descending sorting and switch by clicking on the header). (PRO version only)

Pages fuctionality:

Setting rows number at the page

Free switching between the pages

Serial switching between the pages

Ability to hide/show columns when creating (useful if a field is a key but should not be displayed)

Selective auto-index for columns (index column when creating for faster performance)

Selective auto-filter for columns

Search at the table by substring or regular expression (PRO version only)

Unlimited number of tables on single page, each independently configurable

Custom data types can be plugged in with correct sorting algorithms (PRO version only)

Data definition format is compact and easily generated with server side script (PRO version only)
